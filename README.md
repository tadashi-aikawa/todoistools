todoistools
===========


Prerequirements
---------------

* Python3.7
* Make


### Env

`make init`

### Config

Create `config.yml` and write as following example.

```
token: <todoist_token>
# If you need to
work_project_ids:
  - 1234567890
  - 2234567890
```

Please replace `<todoist_token>` to your todoist token.


Features
--------

### sort

`make sort`
